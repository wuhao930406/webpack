# Ant Design Pro

本项目使用[Ant Design Pro](https://pro.ant.design)进行初始化。以下是如何使用的快速指南。

## 环境准备

安装 `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## Provided Scripts

Ant Design Pro提供了一些有用的脚本来帮助您快速启动和构建web项目，代码风格检查和测试。
`package.json`中提供的脚本。修改或添加其他脚本是安全的:


### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 检查代码风格

```bash
npm run lint
```

### 你也可以使用脚本自动修复一些lint错误:

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
